Лендинг воронки — файлы сюда
============================
Папка: landing/  (рядом с backend, index.html сайта)

ЧТО СЮДА КЛАСТЬ
---------------
  index.html          — главная страница лендинга
  assets/             — картинки, css, js, видео-превью и т.д.
  (или как у вас: css/, img/, js/ — можно своими папками)

Пример:
  landing/index.html
  landing/assets/hero.jpg
  landing/css/style.css
  landing/js/main.js

После загрузки напишите ассистенту: «лендинг залил, посмотри».


ВАЖНО
-----
• Это ОТДЕЛЬНО от сервиса sozdaipesnu.ru (чтобы блок ленда не бил продукт).
• На прод SongForge эта папка сама по себе не выкладывается как сайт.
• Сюда — чтобы разобрать, поправить ссылки, потом выложить на хостинг ленда.


КУДА ПОТОМ ВЫКЛАДЫВАТЬ (простыми словами)
-----------------------------------------
Нужны: 1) новый домен  2) место, где лежат HTML-файлы.

Варианты РФ / дёшево (на выбор):

1) Timeweb / Beget / REG.RU — «хостинг» или «статика», от ~0–150 ₽/мес
   Заливаете файлы из этой папки по FTP/файловому менеджеру.
   Привязываете новый домен. SSL обычно «в один клик».

2) GitHub Pages — бесплатно (серверы не РФ, для ленда часто ок)
   Репозиторий → Pages → папка с index.html.

3) Cloudflare Pages — бесплатно, нужен аккаунт Cloudflare
   Залили папку — получили сайт + можно свой домен.

Рекомендация для старта: новый домен на REG.RU + дешёвый хостинг
Timeweb/Beget «под сайт-визитку» — всё на русском, поддержка РФ.


КНОПКИ НА СЕРВИС (уже в index.html)
-----------------------------------
Все CTA → sozdaipesnu.ru с UTM:
  utm_source=landing
  utm_medium=web
  utm_campaign=gift_song
  utm_content=nav|hero|povody|final

После своего домена ленда — в <head> заменить og:url / og:image
на https://ВАШ-ДОМЕН/ и https://ВАШ-ДОМЕН/assets/og-cover.jpg
(файл assets/og-cover.jpg уже лежит в папке).

Позже: регистрация «с ленда» — отдельная задача.
