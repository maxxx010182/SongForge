=== АКТУАЛЬНОЕ (VK + PKCE фикс) ===

Правки для входа через VK уже отправлены на GitHub.

Чтобы обновить сайт на сервере:

1. Подключись к VPS:
ssh root@195.19.20.245

2. Скачай и примени обновление с GitHub:
curl -fsSL "https://raw.githubusercontent.com/maxxx010182/SongForge/main/scripts/update-now.sh?$(date +%s)" | bash

3. Перезапусти приложение (обязательно с --update-env):
cd ~/SongForge
pm2 restart songforge --update-env

4. Проверь, что VK включён:
curl http://127.0.0.1:8000/api/auth/providers

Должно вывести: "vk_enabled":true

5. На сайте (с Windows):
Открой https://sozdaipesnu.ru
Нажми Ctrl + F5 (полная перезагрузка)
Открой модалку входа
Поставь галочку согласия
Нажми «Войти через VK»

6. Диагностика (если не работает):
pm2 logs songforge --lines 20 --nostream

Ищи строки с vk, /auth/vk, error

Дополнительно:
pm2 status

Если нужно быстро без полного деплоя:
cd ~/SongForge
git pull
pm2 restart songforge --update-env

После обновления:
- Проверь кнопку VK в модалке входа
- Попробуй войти
- Если ошибка — пришли последние строки из pm2 logs (без секретов)
