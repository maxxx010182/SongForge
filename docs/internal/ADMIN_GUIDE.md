## Статус документа

Полное содержание — по плану в `docs/DOCUMENTATION_PLAN.md`. Ниже — краткое резюме из текущей сессии.

### Вход

1. Войти на главной под email из `ADMIN_BOOTSTRAP_EMAILS` в `~/SongForge/.env`
2. `pm2 restart songforge` после изменения `.env`
3. Открыть `/admin` в той же вкладке браузера

### Вкладки

| Вкладка | Назначение |
|---------|------------|
| Обзор | Статистика, зависшие генерации (>10 мин), алерты |
| Генерации | Список и фильтр по статусу (просмотр) |
| Пользователи | Поиск, начисление ± нот с обязательной причиной |
| Команда | Выдача ролей (только super_admin) |
| Журнал | Audit log всех действий |

### Роли

`super_admin` → полный доступ; `support` → пользователи, ±20 нот; `readonly` → только просмотр.

### Пока только через SSH

Восстановление зависших песен: `scripts/check_task.py last --fix` (см. `COMMANDS.txt`).
