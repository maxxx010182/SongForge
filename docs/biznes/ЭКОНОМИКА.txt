Обновлено 15.07.2026 по цифрам владельца.

Статья                    ₽/нота     Источник
------------------------------------------------------------------------
Suno (1 generate = 2 трека)  7,5     15 credits/песня;
                                     1000 credits ≈ 500 ₽ → 15/1000×500 = 7,5 ₽
YandexGPT (plan+lyrics)      3–8     1–4 вызова; оценка ~5 ₽ (уточнить в Cloud)
S3 / диск                    0,3–1   ~5–15 МБ; на бете часто ~0,5 ₽
Брак / retry (+12%)          ~1,5    к (Suno+GPT+S3)
------------------------------------------------------------------------
API «железная» себест.     ≈ 12–16 ₽  (рабочая: ~14,5 ₽)
Эквайринг GetPlatinum        3,5%    от суммы оплаты (факт владельца)
НПД                          4%      от выручки (пока)

ИТОГО на 1 ноту при цене P ₽ (без фикса сервера):

  cost = 14,5 + P×0,035 + P×0,04
       = 14,5 + P×0,075

  margin = P − cost

Примеры (без промо, 1 нота из пакета):
  P=299 → cost≈37 ₽  → маржа ≈ 262 ₽
  P=180 (пакет 10) → cost≈28 ₽ → маржа ≈ 152 ₽

Формула с фиксом:
  доля_фикса = фикс_мес_₽ / платных_нот_мес
  маржа_полная = P − 14,5 − P×0,075 − доля_фикса

4. РЕКЛАМА И ЮНИТ-ЭКОНОМИКА
═══════════════════════════════════════

Чтобы не уйти в минус при трафике:

  CAC = рекламный_бюджет_₽ / число_платящих_клиентов

  LTV = средний_чек_₽ × покупок_на_клиента_за_период

  Правило:  LTV > CAC × 3  (комфортно для масштаба)
  Минимум:  LTV > CAC      (иначе минус)

Планируемый рекламный бюджет:

  Месяц        Бюджет ₽ [ЗАПОЛНИТЬ]    Канал (VK/TG/Директ/блогеры)
  --------------------------------------------------------
  Бета         ___________             ___________
  Запуск       ___________             ___________
  Масштаб      ___________             ___________

Скрытые расходы на трафик (заложить в модель):

  • пробные генерации: ~12–25 ₽ × число_пробных_в_месяц
  • конверсия пробная → покупка: _____ % [измерить на бете]
  • промокоды бета (BETA50): снижают маржу — считать отдельно

Пример (подставить свои цифры):

  100 пробных/мес × 15 ₽ = 1 500 ₽ «маркетинг» без рекламы
  + реклама 10 000 ₽
  + фикс 1 000 ₽
  = 11 500 ₽ затрат до первой продажи

  Нужно продать минимум: 11 500 / (маржа_с_1_ноты) нот
  при марже ~250 ₽/нота → ~46 платных нот только на покрытие.

11. ПРОМО ДЛЯ БЕТА-ТЕСТЕРОВ (чтобы не уйти в минус)
═══════════════════════════════════════

Цель: понять, какую % скидку можно дать на форме GetPlatinum.

Допущения (факт 15.07.2026):
  V ≈ 14,5 ₽  (Suno 7,5 + GPT~5 + S3~0,5 + 12% брак)
  g = 3,5%   GetPlatinum
  n = 4%     НПД
  net = 1 − 0,035 − 0,04 = 0,925

Формула «не в минус по API+GP+НПД» на пакет:

  цена_после_скидки × 0,925  >  V × нот
  цена_после_скидки  >  V×нот / 0,925
  жёсткий пол ₽/нота  ≈ 14,5/0,925 ≈ 16 ₽

  Рабочий пол (запас):     99 ₽/нота
  Комфорт с рекламой:    ≥ 150 ₽/нота

Пересчёт пакетов (V=14,5; g=3,5%; n=4%):

  Пакет    Цена  −30%     −50%     маржа/нота −30%   маржа/нота −50%
  --------------------------------------------------------------------
  1 нота   299   209      150      ~179 ₽            ~124 ₽
  3 ноты   749   524      374      ~147 ₽            ~101 ₽
  5 нот    1199  839      600      ~141 ₽            ~96 ₽
  10 нот   1799  1259     900      ~102 ₽            ~69 ₽

РЕКОМЕНДАЦИЯ ДЛЯ БЕТЫ (по факту Suno 15 cr):

  • Комфорт:     30–40%  — все пакеты с хорошей маржой
  • Агрессивно:  50%     — ок для 10–20 тестеров; notes_10 ~69 ₽/н маржи
  • Не надо:     >60–70% на большом пакете без лимита активаций
  • Бесплатные ноты из админки:  ~15 ₽ × N  (Suno+GPT) — закладывать в маркетинг

Пробные (1/аккаунт): 20 тестеров × ~15 ₽ ≈ 300 ₽ «подарок» API
