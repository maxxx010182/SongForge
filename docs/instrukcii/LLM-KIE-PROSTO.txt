Тексты через Kie.ai (все текстовые задачи)
==========================================
Файл: docs/instrukcii/LLM-KIE-PROSTO.txt
Простыми словами. Код: v2.11.33+


═══════════════════════════════════════
ЗАЧЕМ
═══════════════════════════════════════

• Тексты песен, названия, стиль, помощник — через Kie (не Yandex).
• Музыка Suno — как раньше (sunoapi.org / ApiPass), отдельно.
• Один ключ Kie, две модели: PRO (тексты) и LITE (бот).


═══════════════════════════════════════
ЧТО ПОСТАВИТЬ В .env НА СЕРВЕРЕ
═══════════════════════════════════════

LLM_PROVIDER=kie
KIE_API_KEY=ваш_ключ_с_kie.ai
# KIE_BASE=https://api.kie.ai

# Тексты песен (сильнее)
LLM_MODEL_PRO=gemini-2.5-pro

# Помощник / простые задачи (дешевле)
LLM_MODEL_LITE=gemini-2.5-flash

Потом:
  pm2 restart songforge --update-env


═══════════════════════════════════════
ГДЕ ВЗЯТЬ КЛЮЧ
═══════════════════════════════════════

1) https://kie.ai/api-key — создать ключ
2) Пополнить: https://kie.ai/billing
3) Ключ вставить ТОЛЬКО в .env на сервере (не в чат, не в GitHub)


═══════════════════════════════════════
БАЛАНС В АДМИНКЕ
═══════════════════════════════════════

Откройте https://sozdaipesnu.ru/admin
Блок «Балансы провайдеров»:
  • Kie кредиты — остаток на Kie (тексты)
  • SunoAPI кредиты — остаток sunoapi.org (музыка), если ключ задан
  • Кнопка «Обновить» — перезапросить сейчас

API (для проверки):
  GET /api/admin/provider-balances
  (нужна сессия админа)


═══════════════════════════════════════
ДРУГИЕ МОДЕЛИ (ОПЦИОНАЛЬНО)
═══════════════════════════════════════

Можно сменить slug в .env (path на Kie):
  gemini-2.5-flash
  gemini-2.5-pro
  gemini-3.1-pro
  gemini-3-flash

Не ставьте Opus / gpt-5.5 на поток — дорого.
Claude и часть GPT на Kie — другой формат API; наш клиент заточен
под Gemini openai chat/completions.


═══════════════════════════════════════
ВЕРНУТЬ YANDEX
═══════════════════════════════════════

LLM_PROVIDER=yandex
# и чтобы YANDEX_API_KEY был заполнен
pm2 restart songforge --update-env


═══════════════════════════════════════
ЕСЛИ НЕ РАБОТАЕТ
═══════════════════════════════════════

• 401 — неверный KIE_API_KEY
• 402 — нет кредитов на Kie
• В логах: LLM provider: kie  и  Kie/gemini-... response received
• health: "llm_provider":"kie"
