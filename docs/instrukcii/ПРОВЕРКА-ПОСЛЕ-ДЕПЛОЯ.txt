ЧАСТЬ 1. БЕЗ СЕРВЕРА — только браузер на Windows
------------------------------------------------

1) Откройте сайт:

   https://sozdaipesnu.ru

2) Нажмите Ctrl+F5 (обновить без кеша).

3) Вход по email (сейчас SMTP нет — код на экране):

   - Профиль (иконка человека) → «Войти / Регистрация»
     ИЛИ нажмите «Создать песню» без входа.

   - Должна открыться модалка:
     заголовок «Сейчас начнётся магия» (если задеплоены последние тексты)
     или «Твоя песня — в один клик» / «Вход в аккаунт».

   - Введите свой email → «Получить код».

   - Под полем появится жёлтая строка:
     «Ваш код: 123456 (на почту пока не приходит)»

   - Введите код → «Войти».

   - Должен появиться toast «Вы вошли в аккаунт».
   - В Профиле — ваш email, не «Гость».

4) Telegram и VK:

   Сейчас в .env они ВЫКЛЮЧЕНЫ — кнопок TG/VK в модалке может не быть.
   Это нормально. Работает только email.

5) Админка (если ваш email в ADMIN_BOOTSTRAP_EMAILS):

   https://sozdaipesnu.ru/admin

   Сначала войти на главной под тем же email.


ЧАСТЬ 2. НА СЕРВЕРЕ — по одной строке в SSH
--------------------------------------------

Шаг A. Подключиться:

ssh root@195.19.20.245

(пароль сервера)


Шаг B. Версия сайта:

curl http://127.0.0.1:8000/api/health

Ждём в ответе: "version":"2.11.0"
И желательно: "redis":true, "s3":true


Шаг C. Какие способы входа включены:

curl http://127.0.0.1:8000/api/auth/providers

Сейчас ожидаемо:
  email_enabled: true
  email_smtp: false        ← код на экране, не на почту
  telegram_enabled: false  ← пока нет бота
  vk_enabled: false        ← пока нет VK ID


Шаг D. Redis (у вас уже PONG — можно пропустить):

redis-cli ping

Ответ: PONG


Шаг E. Выйти с сервера:

exit


ЧАСТЬ 3. ЧТО ЗНАЧИТ ОТВЕТ auth/providers
-----------------------------------------

email_smtp: false
  → Войти по email МОЖНО, код показывается на экране в модалке.

telegram_enabled: true (когда настроите)
  → В модалке появится кнопка «Log in with Telegram».

vk_enabled: true (когда настроите)
  → Появится «Войти через VK».
