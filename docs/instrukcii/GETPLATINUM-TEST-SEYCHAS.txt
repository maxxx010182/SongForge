SongForge — тест GetPlatinum на сервере (СЕЙЧАС)
==================================================
Файл: docs/instrukcii/GETPLATINUM-TEST-SEYCHAS.txt
Копировать команды ОТСЮДА (Блокнот), по одной строке.


КОМАНДА 1 — подключиться к серверу
-----------------------------------

ssh root@195.19.20.245


КОМАНДА 2 — перейти в папку проекта
------------------------------------

cd ~/SongForge


КОМАНДА 3 — запустить тест GetPlatinum
---------------------------------------

./venv/bin/python scripts/test_getplatinum.py


ЧТО ПРИСЛАТЬ АССИСТЕНТУ
-----------------------

Скопируйте весь вывод команды 3 (API-ключ в тексте можно заменить на ***).

OK  = formUrl есть — оплата должна открываться.
FAIL = пришлите текст ошибки целиком.
