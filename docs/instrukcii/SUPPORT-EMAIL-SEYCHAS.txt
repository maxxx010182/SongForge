SongForge — ящик support@sozdaipesnu.ru (REG.RU Mail-1)
=========================================================
Файл: docs/instrukcii/SUPPORT-EMAIL-SEYCHAS.txt
Копировать из Блокнота, не из чата Cursor.

Зачем: на сайте вместо личной почты — support@sozdaipesnu.ru
(раздел «Поддержка», футер, форма, юрдокументы).

Почтовый сервер тот же, что для noreply@:
  sm42.hosting.reg.ru, ispmanager, Roundcube
  Подробно: REG-RU-ПОЧТА-ISPmanager.txt (MX уже настроены)


========================================================
ШАГ 1. Создать ящик support@
========================================================

1) REG.RU → Личный кабинет → Почта Mail-1 → Панель ispmanager
2) Почта → Почтовые ящики → Создать

   Домен:     sozdaipesnu.ru
   Имя:       support
   Полный:    support@sozdaipesnu.ru
   Пароль:    придумать и ЗАПИСАТЬ

3) Сохранить.


========================================================
ШАГ 2. Проверка в Roundcube
========================================================

REG.RU → веб-интерфейс Roundcube.

Войти: support@sozdaipesnu.ru + пароль из шага 1.
Если вход работает — ящик создан.


========================================================
ШАГ 3. Куда пересылать (по желанию)
========================================================

В ispmanager у ящика support@ можно настроить:
  • пересылку на вашу личную почту
  • или открывать Roundcube с телефона/ПК

На сервере SongForge (.env) support@ НЕ нужен —
сайт только показывает адрес и mailto: для формы.


========================================================
ШАГ 4. Деплой сайта v2.11.9+
========================================================

После push в GitHub на VPS:

  docs/instrukcii/DEPLOY-SEYCHAS.txt
  (или COMMANDS.txt SHAG 3b)

Проверка в браузере https://sozdaipesnu.ru Ctrl+F5:
  • Поддержка → support@sozdaipesnu.ru (без @m_moshkin и VK)
  • Футер → support@sozdaipesnu.ru
  • /legal/privacy → контактный email support@


========================================================
ШАГ 5. Тест письма
========================================================

1) С любой вашей почты отправьте тест на support@sozdaipesnu.ru
2) Откройте Roundcube → support@ — письмо должно прийти
3) На сайте: Поддержка → форма → «Отправить» — откроется почтовый клиент с адресом support@


Готово? Напишите в чат: «support готов» — перейдём к следующему шагу (TG/VK или оплата).
