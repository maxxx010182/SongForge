JavaScript / TypeScript SDK
Используйте SDK для создания задач, сохранения возвращаемого taskId и получения результатов для Suno.

Установить
Перед запуском этих примеров SDK установите @apipass-dev/apipass-sdk.


Копия
npm install @apipass-dev/apipass-sdk
# or
pnpm add @apipass-dev/apipass-sdk
# or
yarn add @apipass-dev/apipass-sdk
Использование на стороне сервера
Храните свой API-ключ в переменных окружения на стороне сервера. SDK отправляет его в виде токена Bearer и автоматически добавляет заголовки JSON.

Создать задачу
client.jobs.createTask
Создайте новую задачу генерации музыки в Suno V5.

Пример создания задачи с помощью SDK
Воспользуйтесь этим примером, чтобы быстро начать работу с выбранным SDK.


Копия
import { Apipass } from "@apipass-dev/apipass-sdk";

const client = new Apipass({
  apiKey: "apk_79e492a41d440edcc69c5f9e73573b4a77a2fb25e6205356d258ddc3679983b4",
});

const task = await client.jobs.createTask({
  "model": "suno/generate",
  "callBackUrl": "https://your-domain.com/api/callback",
  "input": {
    "prompt": "A catchy pop song about summer love with upbeat tempo and cheerful melody",
    "customMode": false,
    "instrumental": false,
    "vocalGender": "f"
  }
});

console.log(task.data.taskId);

channel
Необязательный
нить
В APIPASS вы можете указать соответствующего поставщика с помощью channelпараметра; эти поставщики обрабатывают фактические задачи генерации изображений и видео. В настоящее время APIPASS предлагает три варианта поставщиков:

Стартовый тариф : Сверхнизкозатратный тариф с ограниченными квотами на использование, невысокой стабильностью и средней скоростью обработки задач.
Обычный : Стандартный уровень, цена которого значительно ниже, чем у официальных API, отличается умеренной стабильностью и средней скоростью выполнения задач.
Официальный : Официальный поставщик, использующий собственный API модели; обеспечивает высокую стабильность, быстрое выполнение задач и цены, идентичные официальному сервису.
Значение параметра по умолчанию channel— auto. При включении APIPASS автоматически распределяет задачи между доступными поставщиками на основе цен в реальном времени и показателей стабильности, чтобы сбалансировать минимальные затраты и надежную работу. Сохраните autoзначение по умолчанию, если у вас нет особых требований к маршрутизации.

Текстовое описание музыки для генерации. Обязательно, если customMode имеет значение false, или если customMode имеет значение true, а instrumental — false.

customMode=false: требуется подсказка, максимум 500 символов. customMode=true + instrumental=false: требуется подсказка. customMode=true + instrumental=true: подсказка необязательна. Для customMode=true: V4: максимум 3000 символов; V5_5, V5, V4_5PLUS, V4_5ALL, V4_5: максимум 5000 символов

"Запоминающаяся поп-песня о летней любви в бодром темпе"

Важные примечания
Каждое поколение создает 2 варианта песни.
Продолжительность песен обычно составляет 2-4 минуты.
Генерация обычно занимает 30-90 секунд.
Аудиофайлы представлены в формате MP3.
Результаты включают изображения обложек.
Задача запроса
client.jobs.recordInfo
Запрос статуса задачи и получение сгенерированной музыки.

Пример SDK для задач запроса
Воспользуйтесь этим примером, чтобы быстро начать работу с выбранным SDK.


Копия
import { Apipass } from "@apipass-dev/apipass-sdk";

const client = new Apipass({
  apiKey: "apk_79e492a41d440edcc69c5f9e73573b4a77a2fb25e6205356d258ddc3679983b4",
});

const result = await client.jobs.recordInfo("task_abc123xyz");

console.log(result.data.state);
console.log(result.data.resultJson);

Поля ответа
code
Код состояния 200 означает успех.

data.taskId
Идентификатор задачи

data.state
Текущее состояние: в очереди, генерация, успех или неудача.

data.resultJson.data
Массив сгенерированных аудиодорожек с URL-адресами и метаданными.

Ошибка - Генерация не удалась, проверьте failCode и failMg.
