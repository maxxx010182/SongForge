Общая картина по самозанятым/НПД подтверждается: чеки нужно формировать через приложение «Мой налог», это единственный способ подтвердить доход и рассчитать налог, и без этого ФНС может назначить штраф за неуплату налогов. Обязанность выдавать чек не зависит от того, как пришли деньги — неважно, наличными, на карту или электронный кошелёк — чек всё равно нужно сформировать.

Теперь конкретно по агрегаторам вроде Гетплатинум — механизм такой же, как у других сервисов для самозанятых: чеки формируются автоматически и уходят в сервис после каждого вывода средств, если подключить интеграцию с приложением «Мой налог» через личный кабинет. То есть логика простая:

**Сам чек убрать нельзя — это требование закона для НПД, а не прихоть конкретного сервиса.** Что меняется с Гетплатинум — только то, кто физически нажимает кнопку "выдать чек". Без интеграции вы делаете это вручную по каждому платежу. С интеграцией — Гетплатинум сам передаёт данные о продаже в «Мой налог», чек формируется автоматически, без вашего участия.

**Важный практический момент, который стоит проверить при подключении:** есть нюанс с корректностью данных при автоматической выгрузке — некоторые сервисы могут неправильно определять поступления как доход от физического лица, из-за чего налог считается по ставке 4% вместо правильной 6% для дохода от юрлиц/ИП (в вашем случае, скорее всего, применяется ставка для физлиц-покупателей — 4%, раз клиенты обычные пользователи, но сам факт возможной ошибки в автоматическом определении ставки стоит перепроверить на первых порах). Рекомендация из практики — первое время сверять автоматически сформированные чеки вручную в приложении, пока не убедитесь, что интеграция считает верно.

Так что для документов ничего менять не нужно — пункт "чек направляется через Мой налог" остаётся верным и с Гетплатинум, просто фактически станет происходить в фоне, а не вашими руками.
