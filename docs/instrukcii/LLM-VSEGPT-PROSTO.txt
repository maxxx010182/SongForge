Тексты и боты: разные модели с одного ключа (VseGPT / OpenAI-compatible)
======================================================================
Файл: docs/instrukcii/LLM-VSEGPT-PROSTO.txt
Простыми словами. Код: v2.11.32+


═══════════════════════════════════════
ЗАЧЕМ
═══════════════════════════════════════

• Тексты песен — нужна СИЛЬНАЯ модель (красивый русский, структура).
• Помощник на сайте / простые задачи — можно ДЕШЁВУЮ и быструю.
• Один API-ключ (VseGPT), в запросе меняется только «какая модель».

По умолчанию сайт по-прежнему на Yandex, пока не переключите.


═══════════════════════════════════════
КАК УСТРОЕНО В КОДЕ
═══════════════════════════════════════

Роль                Логика          Модель (по умолчанию для VseGPT)
-------------------------------------------------------------------
Тексты песен        MODEL_PRO       deepseek/deepseek-v4-pro
Помощник, план      MODEL_LITE      deepseek/deepseek-v4-flash
Style / analyst     PRO или LITE    по вызову в коде

Меняете id моделей в .env — без переписывания всего сайта.


═══════════════════════════════════════
ВКЛЮЧИТЬ VseGPT (когда будет ключ)
═══════════════════════════════════════

1) Зарегистрироваться на https://vsegpt.ru/ , пополнить баланс (рубли).
2) Создать API-ключ.
3) На сервере в ~/SongForge/.env добавить (или заменить):

LLM_PROVIDER=openai_compat
OPENAI_COMPAT_API_KEY=sk-...ваш_ключ...
OPENAI_COMPAT_BASE=https://api.vsegpt.ru/v1
LLM_MODEL_PRO=deepseek/deepseek-v4-pro
LLM_MODEL_LITE=deepseek/deepseek-v4-flash

4) pm2 restart songforge --update-env

5) Тест: создать песню + спросить помощника.

Вернуться на Yandex:
LLM_PROVIDER=yandex
(и чтобы YANDEX_API_KEY был заполнен)


═══════════════════════════════════════
ПОДБОР МОДЕЛЕЙ (игра)
═══════════════════════════════════════

Каталог id: https://vsegpt.ru/Docs/Models

Примеры для PRO (тексты):
  deepseek/deepseek-v4-pro
  deepseek/deepseek-v4-flash   (если pro дорого, flash часто хватает)
  openai/gpt-4o-mini
  google/gemini-2.5-flash

Примеры для LITE (бот):
  deepseek/deepseek-v4-flash
  google/gemini-2.5-flash-lite
  openai/gpt-4o-mini

Правило: 2–3 тестовые песни на новой PRO, потом оставляете лучшую.


═══════════════════════════════════════
ЦЕНЫ (ориентир VseGPT)
═══════════════════════════════════════

DeepSeek V4 Flash — копейки за песню (текст).
DeepSeek V4 Pro — дороже flash, всё ещё обычно дешевле «толстого» Yandex Pro.
Suno (музыка) — отдельно, не через VseGPT.


═══════════════════════════════════════
ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
═══════════════════════════════════════

• 401 — неверный ключ
• 402 / нет баланса — пополнить VseGPT
• Неверное имя модели — сверьте id на Docs/Models
• В логах: OpenAI-compat/... response received
