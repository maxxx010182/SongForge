# SongForge — правила для ассистента

Папка: `C:\Users\makc0\Desktop\SongForge`

## Перед любой работой прочитать

1. `SONGFORGE-КОНТЕКСТ.txt` — **главный handoff**: текущий статус, следующий шаг, тексты UI
2. `docs/NOTES_NEXT.md` — технический backlog (верх файла = актуально; ниже — архив)
3. `COMMANDS.txt` — команды для сервера (копирует владелец)

Не опираться на старые цифры версий в середине длинных файлов — только на блок **«СЕЙЧАС»** в `SONGFORGE-КОНТЕКСТ.txt` и на `backend/app.py`.

## Команды для сервера

Файл `COMMANDS.txt` (латиница). Пользователь копирует оттуда — из чата Cursor копирование на Windows часто не работает.

Инструкции — только `.txt`, без `.bat`-обёрток (пользователь открывает текстовые файлы сам).
